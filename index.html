关于hexo

发表于 2021-04-28

一、 什么是Hexo ?

  • Hexo 是一个快速、简洁且高效的博客框架
  • Hexo 使用 Markdown (或其他渲染引擎) 解析文章，几秒内，即可利用靓丽的主题生成静态网页

二、如何安装 ？

1、安装：只需要几分钟时间

2、安装前提：

  • Node.js (建议版本 node.js > 12.0)
  • Git

3、安装Git

  • Windows :下载并安装Git
  • Mac : 使用Homebrew
  • Linux(Ubuntu、Debian): sudo apt-get install git-core
  • Linux(Fedora、Red Hat、CentOS): sudo yum install git-core

4、安装Nodejs

  • Windows ：通过 nvs（推荐）或者nvm 安装。
  • Mac ：使用Homebrew
  • Linux(DEB/RPM-based) ：从 NodeSource 安装

5、安装Hexo ：npm install hexo

6、安装完成，两种方式执行

  • npx hexo
  • 将 Hexo 所在的目录下的 node_modules 添加到环境变量之中即可直接使用 hexo ：

7、Nodejs版本限制

  • 强烈建议永远安装最新版本的 Hexo，以及 推荐的 Node.js 版本。
